React Create Component

A command line interface for building React components.

Installation

As a global dependency

$ yarn add rcc-cli --global
$ npm install rcc-cli -g

React Create Component ships with a executable file that binds to the command: rcc.

Usage

RCC accepts three parameters: type, name, and path.

$ rcc [type] [name] [path]

Type: [Required] the type of component you wish to create. Container, Component, or Pure.

Name: [Required] the name of the component to be created. This will not only be the name of the newly created file, but will also map to several instances inside of the component itself.

Path: [Optional] the directory path from cwd() that you wish to create a component. If not provided, it will be created in your current working directory.

Example

$ rcc container HeaderContainer src/containers

will generate the following file in cwd()/src/containers

  state = {};

    const props = {};

    );
  }
}

HeaderContainer.displayName = 'HeaderContainer';

HeaderContainer.propTypes = {};

function mapStateToProps(state) {
  const { example } = state;
  return {
    example: example.examples,
  };
}

function mapDispatchToProps(dispatch) {
  return {
    example: bindActionCreators(ExampleActions.exampleFunction, dispatch),
  };
}

export default connect(mapStateToProps, mapDispatchToProps)(HeaderContainer);
